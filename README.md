# Kutira-Kushala

Kutira-Kushala is an Android-based artisan marketplace application designed to connect local village craftsmen and buyers through a modern digital platform. The application helps artisans showcase handmade products, communicate directly with customers, manage bulk orders, and grow their business digitally.

---

## Features

- User Authentication (Login & Registration)
- Seller Profile Management
- Product Upload with Multiple Images
- Real-time Buyer & Seller Chat
- Bulk Order Request System
- Notifications & Unread Message Badge
- Product Catalog Browsing
- WhatsApp & Call Integration
- Firebase Cloud Backend
- Real-time Firestore Updates
- Modern Android UI Design

---

## Technologies Used

### Frontend

## Project Objective

The main objective of Kutira-Kushala is to support rural artisans and small-scale craftsmen by providing a digital marketplace where they can directly connect with buyers, sell handmade products, and manage business operations efficiently.
